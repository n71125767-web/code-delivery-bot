# V56 Clean Stability Fix

Исправления:

- добавлен PostgreSQL advisory-lock против двух polling-инстансов Telegram;
- убраны некорректные параметры `style` у inline-кнопок;
- исправлено пересечение кнопки «⚙️ Админ меню» с настройками/FAQ;
- очищены админ/поставщик переходы и скрытые разделы;
- прокси-меню покупателя упрощено до нужных кнопок;
- страны прокси отображаются по-русски через существующий `country_ru`;
- «Мои заказы» показываются кнопками по 3 заказа на страницу;
- товары без категории доступны покупателю через «📂 Без категории»;
- расширенные настройки товара открываются на карточке товара, не отдельным пустым экраном;
- удалённые товары можно физически очищать из админки;
- проект очищен от `__pycache__` и старых patch-notes.

Важно: конфликт `getUpdates` полностью исчезает только если все процессы используют одну БД. Если где-то запущен второй сервис с тем же BOT_TOKEN и другой DATABASE_URL, его всё равно нужно остановить.
