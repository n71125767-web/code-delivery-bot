# PATCH V51 — UI, roles, orders, products

## Исправлено
- Товар можно выдать покупателю по Telegram ID или `@username`.
- `@username` ищется по пользователям, которые уже открывали бота.
- «Мои заказы» теперь показываются страницами: максимум 3 покупки на одно сообщение, кнопки «Назад / Вперёд».
- Удаление товара из V25 больше не оставляет его в витрине: бот пытается удалить строку физически, а если товар уже связан с покупками, полностью скрывает его из каталога и админских списков.
- Товар можно создать и опубликовать без категории.
- В мастере создания товара добавлена кнопка «Без категории».
- Кнопка «Я поставщик» скрыта для покупателей, которые не являются поставщиками.
- Панель поставщика больше не запускает старую рассылку, если у админа остался незавершённый broadcast-state.
- Админ-панель больше не спамит отдельным сообщением «Админ-панель» при нажатии назад.

## Визуал
- Переработаны главные кнопки покупателя, поставщика и администратора.
- Добавлены эмодзи в админ-меню.
- Переработаны визуалы разделов: настройки, статистика, скрытые, способы оплаты, кошелёк.
- Админские страны прокси показываются сеткой с пагинацией, а не одним длинным столбиком.
- Переработан визуал прокси-настроек для администратора.

## Администраторы и права
- Добавлен раздел «Админы и права».
- Главный админ/ГА может выдавать и убирать доступы дополнительным админам по разделам:
  - товары;
  - оплата;
  - настройки;
  - рассылка;
  - прокси;
  - статистика;
  - админы;
  - скрытые;
  - поставщики.

## FAQ и главная
- FAQ и главная страница теперь редактируются через кнопки в админке:
  - Админ → Настройки → Изменить FAQ;
  - Админ → Настройки → Изменить главную.

## Команды
Слеш-команды оставлены как резервный способ управления, но основные действия вынесены в кнопки.
