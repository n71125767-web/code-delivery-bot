# MCS Visual V32

Обновление реального Telegram-бота по присланным скриншотам.

Изменено:
- категории показываются медиа-карточками с фото, описанием и кнопками снизу;
- товары показываются карточками с фото/видео, ценой, описанием и кнопкой CryptoBot;
- если у категории нет загруженного фото, используются встроенные баннеры для номеров, инструментов, подтверждений, маркета, разного и прокси;
- кнопки списка товаров показывают название без лишнего префикса и цены, как в референсе;
- главная витрина начинается с текста «Выберите товар:»;
- кнопка покупки переименована в «🤖 Оплатить • CryptoBot»;
- сохранено управление фото и видео из админ-панели;
- сохранены собственный магазин, Crypto Pay, Proxyline, поставщики и номера;
- Admaker не используется.
