# V53 Clean Visual Hotfix

## Исправлено

- Убран дублирующий список товаров текстом в карточке категории: товары теперь идут кнопками.
- Цена форматируется компактно: `100`, `100.1`, `100.10`, без `100.00000000`.
- В админском каталоге категории и товары вынесены в кнопки.
- Вернул `🌐 Прокси` в главное меню покупателя.
- Главное меню покупателя перестроено:
  - `🛒 Товары` / `🛍 Корзина`
  - `📱 Номера` / `🌐 Прокси`
  - `💼 Кошелёк` / `🧾 Мои заказы`
- Убрана кнопка зеркал из скрытого раздела.
- Раздел `🧩 Прокси` в админке очищен: нет отдельной кнопки `Товары`, товары настраиваются через `Управление товарами`.
- В тексте наценки прокси убраны `/`-команды.
- Раздел `📱 Номера` больше не подмешивает обычные партнёрские товары только потому, что они привязаны к поставщику.
- Панель поставщика сокращена и очищена от длинной справки.
- При отмене действия поставщик возвращается в поставщик-клавиатуру, а не в админ-меню.
- Убран лишний развёрнутый текст из админ-панели и панели поставщика.
