# V69 Supplier Assignment / Spam Fix

Исправления:

- Убрано срабатывание рассылки/активной заявки при нажатии кнопок панели поставщика.
- Кнопки поставщика теперь обрабатываются как навигация панели, а не как номер/код.
- Добавлена кнопка `🚚 Поставщик` в карточку товара администратора.
- При назначении поставщика товару админ вводит Telegram ID, после чего пользователь автоматически становится активным поставщиком.
- Товар привязывается к поставщику по `product_id`, `internal_key` и legacy `ProductProvider`, чтобы покупки и заявки доходили поставщику.
- Поставщику приходит уведомление о назначении на товар.
- Одобренные marketplace-заявки теперь также привязывают поставщика по `product_id` и `internal_key`, чтобы заявки по покупкам не терялись.

Проверка:

```bash
python -m compileall -q app
```
