# V46 — Clean UI / Admin Mode / Hidden Proxy Category

Что изменено:

- Убран показ HTML-тегов `<b>`, `<i>`, `<code>` и похожих тегов в сообщениях. Тексты теперь уходят в Telegram как обычный читаемый текст.
- Главное меню покупателя очищено: кнопка «Прокси» убрана с главной панели.
- При входе в админ-панель бот убирает нижнюю покупательскую клавиатуру, чтобы не смешивать режимы покупателя и администратора.
- Кнопка «Главная» из админки возвращает в покупательскую панель.
- Админ-панель стала мягче визуально: кнопки в одном стиле, без смешивания с покупательскими кнопками.
- Исправлен раздел «Скрытые»: теперь это отдельное меню с кнопками, а не просто текст.
- Названия провайдеров прокси убраны из пользовательских текстов и основных админ-подсказок. Покупатель видит только «прокси» и данные подключения.
- Системная категория «Прокси» больше не создаётся в главном каталоге через `/proxy_autofix`.
- Старая системная категория «Прокси» автоматически скрывается при `/proxy_autofix` и вручную через `/hide_proxy_category`.
- Прокси-товары отвязываются от главных категорий, чтобы не появляться в каталоге с картинкой категории.
- Cooldown на кнопки убран для админов и поставщиков.
- Для покупателей cooldown оставлен только на рискованные действия: проверка оплаты, подтверждения, оформление/повторные действия. Обычная навигация без задержки.

Новая команда:

```text
/hide_proxy_category
```

Скрывает старую системную категорию «Прокси» из главного каталога и отвязывает системные прокси-товары от категорий.

После деплоя рекомендуется выполнить:

```text
/hide_proxy_category
/proxy_autofix 100 RUB
/proxy_markup 1.77
```
