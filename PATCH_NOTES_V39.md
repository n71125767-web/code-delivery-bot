# V39 — Full visual + proxy markup

## Что изменено

- Переработано главное меню покупателя: компактные кнопки и понятные подписи.
- Переработана админ-панель: основные разделы сверху, редкие действия спрятаны в `🕶 Скрытые`.
- Нижняя reply-клавиатура администратора теперь показывает только один вход `🛠 Админ`, без лишних опасных кнопок.
- Админ-кнопки не показываются обычным покупателям.
- Раздел прокси стал чище: тип → страна / поиск → срок → оплата.
- Добавлена автоматическая наценка на прокси: по умолчанию `×1.77`.
- Базовая цена хранится в товаре, покупатель видит цену `база × наценка`.

## Новые команды

```text
/proxy_markup
```
Показать текущую наценку.

```text
/proxy_markup 1.77
```
Изменить коэффициент наценки.

```text
/proxy_price 100 RUB
```
Изменить базовую цену всех Proxyline-товаров. При наценке `×1.77` покупатель увидит `177 RUB` за 1 месяц.

```text
/proxy_autofix 100 RUB
```
Создать/обновить стандартные Proxyline-товары. Цена в команде — базовая, к ней применяется наценка.

## Настройки Render

Можно задать дефолтную наценку через env:

```text
PROXY_MARKUP_MULTIPLIER=1.77
```

После изменения через команду `/proxy_markup` значение хранится в базе и имеет приоритет для бота.
